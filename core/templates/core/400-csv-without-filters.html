Oops! Existem melhores formas de baixar o CSV completo de um dataset

Fazer o download de um CSV pela interface web onera muito nossos servidores. Como isso influencia na experiência usuários, essa requisição foi invalidada.

Se você precisa baixar a base completa, por favor, baixe o arquivo com dataset completo.

Esse link usa nosso servidor de arquivos estáticos e é a maneira mais rápida (e correta) de baixar a base de dados completa. Por favor, não use a API para percorrer todas as páginas do dataset, nem tente baixar com o "?format=csv" do link de filtros que aparece na interface Web pois ambos oneram demais nossos servidores.
